Files and assets used to create Evolution Monsters:

	- Official Attribute icon .SVG files, from yugipedia.com.
	- Card images exported from https://www.cardmaker.net/yugioh.
	- Card frame assets and fonts used by https://www.cardmaker.net/yugioh, available at https://www.cardmaker.net/uploads/cardmaker_resources/.
	- Fonts used by https://lauqerm.github.io/ygocarder/, available at https://github.com/lauqerm/ygocarder/tree/main/public/asset/font.
	- Reddit user u/Ray_V_Xyz's Evolution Monster frame as reference, showcased at https://www.reddit.com/r/customyugioh/comments/ob6p7t/custom_summoning_mechanic_evolution_summoning/, available at https://www.deviantart.com/stash/01n9ptjh1b99, with required fonts available at https://www.deviantart.com/stash/0178kmbr0yde. None of these assets from u/Ray_V_Xyz were directly used to create my Evolution Monster frame.
